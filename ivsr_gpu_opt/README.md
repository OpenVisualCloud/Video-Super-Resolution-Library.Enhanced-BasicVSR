# iVSR GPU Opt
This folder enables the inference of BasicVSR (an AI-based VSR algorithm) on GPU (Data Center GPU Flex series, aka. ATS-M)
with OpenVINO. 

| Version | Description |
|---------| ----------- |
| ov_1    | Based on OpenVINO 2022.1.0       |
| ov_3    | Based on OpenVINO 2022.3.0 (LTS)       |
